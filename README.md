Profile

I design and operate the infrastructure layer that production depends on — systems engineered to fail small, recover fast, and stay within their error budget under real-world conditions: traffic spikes, partial outages, bad deploys, and infrastructure drift happening at the same time.

My work sits at the intersection of cloud architecture, automation, and incident response — reducing manual intervention, shrinking blast radius, and turning operational knowledge into code rather than tribal memory.
